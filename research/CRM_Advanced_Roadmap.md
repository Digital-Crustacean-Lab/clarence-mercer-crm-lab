# CRM Advanced Analytics Roadmap: The Mercer Masterclass

## Phase 4: Customer Lifetime Value (CLV) Mastery
- **Objective**: Predicting the future value of a customer based on historical transaction density.
- **Key Knowledge**: BG/NBD Model (Buy Till You Die), Gamma-Gamma Submodel.
- **Dataset**: `Online Retail` (Extended) or `bhanupratapbiswas/customer-lifetime-value-analytics-case-study`.
- **Status**: Ready to start.

## Phase 5: CRM Anomaly & Fraud Detection
- **Objective**: Identifying high-risk behaviors like return abuse or account takeovers.
- **Key Knowledge**: Unsupervised Learning (Isolation Forest), Outlier Detection.
- **Dataset**: `computingvictor/transactions-fraud-datasets`.
- **Status**: Pending Phase 4.

## Phase 6: Multi-Touch Attribution (MTA) & Shapley Value
- **Objective**: Quantifying the contribution of each marketing channel to a single conversion.
- **Key Knowledge**: Game Theory (Shapley Values), Markov Chain Attribution.
- **Dataset**: `olistbr/marketing-funnel-olist`.
- **Status**: Pending Phase 5.
